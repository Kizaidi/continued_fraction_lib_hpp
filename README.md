# Библиотека цепных дробей на C++

## Описание

Профессиональная реализация алгебраического объекта "Цепная дробь" на языке C++.
Библиотека поддерживает символьные вычисления, алгебраические операции, упрощение
и ввод/вывод данных.

## Особенности

1. **Полная реализация алгебраического объекта:**
   - Создание/удаление объектов
   - Копирование и присваивание
   - Определение свойств объекта

2. **Алгебраические операции:**
   - Сложение, вычитание, умножение, деление
   - Составные операторы (+=, -=, *=, /=)
   - Операторы сравнения

3. **Символьные вычисления:**
   - Преобразование из/в строку
   - Парсинг различных форматов
   - Упрощение выражений

4. **Численные методы:**
   - Подходящие дроби (конвергенты)
   - Преобразование в double
   - Вычисление с заданной точностью

5. **Специальные функции:**
   - Цепные дроби для √n, e, π
   - Периодические дроби
   - Рациональные приближения

## Требования

- Компилятор с поддержкой C++20
- CMake 3.10+

mkdir build
cd build
